## E-Commerce Sales Analysis – Excel Project

### 1. Project Title
E-Commerce Sales Analysis using Excel

### 2. Objective
To analyze sales performance across products, categories and cities to identify top-performing areas and business trends.

### 3. Dataset Description
The dataset contains e-commerce order details including Order ID, Product, Quantity, Price, Order Date, City, Category and Total Amount.

### 4. Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts

### 5. Analysis Performed
- Top 5 products by sales
- Category-wise sales analysis
- City-wise category performance
- Day-wise sales trend analysis

### 6. Key Insights
- Laptop category performs highest in cities like San Francisco and Boston.
- Mobile category dominates in Los Angeles and New York City.
- Monitor category shows strong demand in Dallas and Atlanta.
- Accessories contribute steady sales across all cities.


### Project Highlights
- Category-wise, City-wise and Day-wise sales analysis
- Top 5 Products and Top Cities identification
- Data cleaning and category mapping using VLOOKUP
- Interactive Pivot Tables and Charts for visualization
